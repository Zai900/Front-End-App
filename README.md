# Zainab_backend

This is the **backend API** for my CST3144 Full Stack Development coursework,  
_After-School Activity Club_.

It is built with **Node.js**, **Express**, and **MongoDB Atlas** and provides
REST endpoints for:

- Listing lessons (subject, location, price, spaces, image)
- Searching and sorting lessons (handled in the frontend)
- Creating orders with lesson quantities
- Updating lesson spaces after orders
